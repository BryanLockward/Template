Allwebco Template

This template includes three picture galleries in 2 versions, both with and without Paypal order forms. The header logo is a global plain text logo you can edit in Notepad. A graphic logo .jpg option is inlcuded so you can use your own graphic logo image. All menus, sidebars and the footer and header are plain text global files so you can update all your pages at one time. The included search is a site search powered by Google and does not require any special scripts to be setup in your hosting account. View this page with a graphic logo.

Colors and Formatting

This template includes six color themes. Colors are easily, globally changed for the entire template by editing the plain text colortheme.css with your choice. Instantly change the color of your website. Template font sizes and colors, page buttons and menu colors and fonts are all set in the global CSS files for easy formatting updates. You can also setup your own custom color them. If you are creating your own theme some graphic editing will be required.

Features

Plain text global menus, global sidebar areas, header and footers are setup for easy website updates. The drop menus can be setup 3 deep for leager websites. A sample multi-level menu file is included with the download. Or any menu item can be set as a plain button with no drop menu. The sidebar search can be switched off, setup as a Google search or, 3 options are inlcded for a site search. The sidebar scroller is displaying 3 basic HTML pages and is updated like any other template page. The rihgt sidebar can be switched off of not required. View the step by step instructions.
